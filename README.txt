Adds the Facebook Pixel to your site.

== Description ==

> <strong>Facebook Pixel</strong><br>
The Facebook Pixel enables you to track the performance of your Facebook advertisements.

= SMNTCS Facebook Pixel =
SMNTCS Facebook Pixel enables you to add the Facebook Pixel to your website.

== Installation ==

1. Upload 'smntcs-facebook-pixel' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to https://www.facebook.com/business/help/952192354843755?helpref=faq_content#createpixel to create a Facbook Pixel
4. Go to 'Appearance' » 'Customize' » 'Facebook Pixel' and paste your Facebook Pixel code

== Frequently Asked Questions ==

= Why do I need the Facebook Pixel in first place? =

You run Facebook advertisements, but you don't know how they performe? SMNTCS Facebook Pixel enables you to track the progress of your advertisements.

= Why am I not able to save the Facebook Pixel code? =

This issue might be caused by a security plugin. If you use a security plugin, e.g. Wordfence, then disable it so save your Facebook Pixel code and activate it once you’re done.
